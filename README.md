 MyChat — REST API Chat Application

A simple chat backend built with Java Spring Boot and MySQL. Users can register, 
log in, and send/receive text messages through a REST API. Includes full CRUD 
operations for messages, a custom profanity filter, input validation, and proper 
exception handling with clean error responses. Built using a layered architecture 
(Controller → Service → Repository) with Spring Data JPA, and tested with JUnit. 
A simple HTML/JavaScript frontend connects to the API for sending and viewing messages.

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- JUnit
- HTML/CSS/JavaScript
